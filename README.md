## Conceptos

Contenedor
```html
<!-- Soy un contenedor -->
    <div class="contenedor"> 
<!--  -->
        <div class="item item1">Item 1</div>
        <div class="item item2">Item 2</div>
    </div>
````
Item
```html
    <div class="contenedor"> 
        <!-- Soy un item -->
        <div class="item item1">Item 1</div>
        <!--  -->
        <div class="item item2">Item 2</div>
    </div>
```
